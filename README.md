# CAS Applied Data Science 2025

The CAS program at the University of Bern is structured into 6 modules.

The CAS ADS program consists of the following modules

## Module Overview

**Modul 1:** Data Acquisition and Management

**Modul 2:** Statistical Inference  for Data Science

**Modul 3:** Data Analysis and Machine Learning

**Modul 4:** Ethics and Best Practices

**Modul 5:** Consolidations

**Modul 6:** Deep learning
